# Christmas Tree Website

This project is a web application featuring a Christmas Tree on the main page. Visitors can click on the tree to craft a toy and provide wishes. The application uses Entra ID for Single Sign-On (SSO) authentication and is hosted on Azure WebApp.

## Features
- **Interactive Christmas Tree**: Visitors can click on the tree to craft a toy and provide wishes.
- **SSO Authentication**: Secure authentication using Entra ID.
- **Azure Hosting**: Deployed on Azure WebApp for scalability and reliability.

## Setup Instructions
1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd ChrismasTree
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run Locally**:
   ```bash
   npm run dev
   ```

4. **Deploy to Azure**:
   Follow the deployment instructions in the `DEPLOYMENT.md` file.

## Project Structure
- `src/`: Contains the source code for the application.
- `public/`: Static assets like images and icons.
- `README.md`: Project documentation.
